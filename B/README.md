### 主要参数与影响

- AutoEncoder: 
  - 训练轮次: 10次收敛
  - 学习率: 0.2最佳
  - droput: 0.1最佳
- NCF: 
  - factor_num (EMB_SIZE): 大于等于10时对模型效果无明显影响，但会加长训练时间
  - num_layers: 神经网络层数，在较小数据上设为1层有比较好的效果
  - droput: 0.2最佳
